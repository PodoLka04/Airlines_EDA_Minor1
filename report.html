Вопросы

Вопрос 1

Вопрос: -Правда ли оценка ширины кресла (seat_width) выше, если перелет был в летнее время?

Данные: для ответа на вопрос нужны таблицы seat.

seat_with_summ = seat %>% mutate(summ = (month(date_flown) %in% c(6,7,8)))
ggplot() +
  geom_boxplot(data = seat_with_summ, aes(x = factor(summ), y = seat_width_rating), col = "#E52B50", fill = "#F0F8FF") +
  xlab("Перелет в летний месяц?") +
  ylab("Оценка ширины кресла") +
  ggtitle("Распределение оценок ширины кресла в зависимости\n от времени года") 

mean_for_summ = seat_with_summ %>% filter(summ) 
a = mean(mean_for_summ$seat_width_rating)
cat(a," средняя арифметическая оценка летом \n")
## 2.625  средняя арифметическая оценка летом
mean_for_notsumm = seat_with_summ %>% filter(summ == F) 
b = mean(mean_for_notsumm$seat_width_rating)
cat(b," средняя арифметическая оценка не летом")
## 2.814286  средняя арифметическая оценка не летом

Ответ: Нет, в летние месяцы оценка ширины кресел ниже, чем в другие месяцы.

Вывод: На основе исследования оценки ширины кресел в зависимости от времени года, видно, что и среднее арифметическое, и медианное значение оценки летом ниже, чем не летом. Кроме того, из анализа при помощи ящика с усами следует, что летом половина людей ставит оценку 2 и ниже, а в иное время года половина пассажиров за 3 и ниже, другая же половина, за 3 и выше. То есть летом недовольных качеством ширины сидения больше. Нужно прислушатся к отзывам о ширирне кресел в летнее время и попытаться скрыть этот недостаток. Необходимо либо фактически изменить сидения, либо посмотреть корреляцию оценки некоторого иного признака с оценкой ширины сидения, и влиять на оценку ширины при помощи изменения этого признака. Кроме того, внимания заслуживает факт, что в общем и целом оценки в обоих случаях достаточно низкие, т.е. нужно пытаться изменить и улучшить кресла.

Вопрос 3

Вопрос: -Какие лаунж-зоны оценены по чистоте (cleanliness) выше, чем аэропорты, в которых они расположены?

Данные: для ответа на вопрос нужны таблицы lounge, airport.

loun = lounge %>% select(title = airport, cleanliness_rating,lounge_name)
loun$lounge_name = str_remove_all(loun$lounge_name, " LOUNGE REVIEW")
loun$lounge_name = str_remove_all(loun$lounge_name, " CUSTOMER REVIEW")

airp = airport %>% select(title, terminal_cleanliness_rating) 
airp$title = str_remove_all(airp$title, " customer review")
airp$title = str_remove_all(airp$title, " CUSTOMER REVIEW")


vmeste = inner_join(loun, airp)  %>% filter(!is.na(terminal_cleanliness_rating))%>% filter(!is.na(cleanliness_rating))  %>% group_by(lounge_name) %>% summarise(mean_terminal_cleanliness_rating=mean(terminal_cleanliness_rating), mean_lounge_cleanliness_rating = mean(cleanliness_rating)) %>% filter(mean_lounge_cleanliness_rating>mean_terminal_cleanliness_rating)

otv = left_join(vmeste, select(loun, title, lounge_name)) %>% distinct(.keep_all = TRUE ) %>% mutate(raznitsa =mean_lounge_cleanliness_rating - mean_terminal_cleanliness_rating ) 
otv$full = paste(otv$title, otv$lounge_name, sep=" \n")




q = top_n(otv,5,otv$raznitsa) %>% arrange(-raznitsa) #смотрим полезную информацию - топ 5 аэропортов с большей разницей двух оценок, из otv - переменная q


ggplot()+
  geom_bar(data = q, aes(x = full, width = 0.8, y =mean_lounge_cleanliness_rating), stat = "identity", col = "#403a3a", fill = "pink") + 
  geom_bar(data = q, aes(x = full,width = 0.6, y = mean_terminal_cleanliness_rating), stat = "identity", col = "grey", fill = "#9224A6", alpha = 0.5)+ 
  theme(axis.text.y = element_text(angle =30,size = 6))+
  xlab("Назавние аэропорта и лаунж зоны") +
  ylab("Оценка чистоты аэропорта(пурпурный) \n Оценка чистоы лаунж зоны(розовый)") +
  ggtitle("Топ 5 аэропортов с наибольшей \n разницей чистоты,\n удовлетворяющих условию")+
  coord_flip()

Ответ: Все аэропорты вместе с названиями лаунж зон, удовлетоврящие условию находятся в переменной otv(их 217)

Вывод: Кроме списка из аэропортов, где оценка чистоты лаунж зоны выше оценки общей чистоты аэропорта(переменная otv), приведен топ 5 аэропортов c конкретными лаунж зонами в них(так как на один аэропорт может быть несколько лаунж зон), удовлетворяющих условию, где разрыв оценок наибольший(На графике это то, насколько розовый цвет(от 0) длиннее пурпурного). Нужно обратить внимание тех персонала данных аэропортов на их приоритет лаунж зон и потреобовать большей чистоты и вне лаунж зон. Возможно это означает избыток персонала в лаунж зонах и недостаток их в других местах данного аэропорта. Исходя же из всего списка, необходимо провести работу с тех персоналом в данных аэропортах, замотивировав их лучше выполнять свою работу, например использовать систему бонусов.
